# air-quality-monitoring-system

## 📌 Description
This project monitors air quality using ESP8266, MQ-135, and DHT11 sensors.

## ⚙️ Features
- Real-time monitoring using Blynk
- Temperature and humidity tracking
- LCD display output

## 🛠️ Components
- ESP8266
- MQ-135 Gas Sensor
- DHT11 Sensor
- 16x2 LCD

- Add mobile alerts
- Improve accuracy
